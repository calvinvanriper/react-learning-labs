# React Learning Labs

A structured React learning workspace for course projects, component drills, practice labs, and mini-projects.

## Purpose

This repository is for learning React fundamentals and building small practice projects before moving into larger portfolio-ready React applications.

## Structure

```bash
react-learning-labs/
  course-projects/
  practice-labs/
  mini-projects/
```

## Current Focus

- React fundamentals
- Components
- Props
- State
- Events
- Conditional rendering
- Lists
- Controlled forms
